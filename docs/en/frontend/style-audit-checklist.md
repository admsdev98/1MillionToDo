# Style audit checklist (EN)

Goal: align the assessment UI with the 1millionbot.com design direction without copying the site.

## Typography
- [ ] Confirm body font and weights
- [ ] Confirm heading font and weights
- [ ] Define fallbacks

## Colors
- [ ] Confirm ink (primary text)
- [ ] Confirm accent and CTA
- [ ] Confirm hover/active/disabled
- [ ] Define error/success colors

## Components
- [ ] Buttons: pill radius, padding, border
- [ ] Inputs: border, focus ring
- [ ] Cards: radius, shadow
- [ ] Badges: OWNER/SHARED

## Layout
- [ ] Spacing scale (8/16/24/32)
- [ ] Max width rules (forms/dashboard)
- [ ] Mobile breakpoints

## Motion
- [ ] One subtle page fade
- [ ] Optional list stagger

## Output
- [ ] Write final tokens and patterns to `style-tokens-and-ui-patterns.md`
